ProxmoxKit

🚧 Preview Release

A modern Swift library for interacting with the Proxmox Virtual Environment API. Built with service-oriented architecture and comprehensive async/await support.

📚 View Documentation 💻 GitHub Repository

🚀 Modern Architecture

  • Service-oriented design for clean API organization
  • Full async/await support for all operations
  • Type-safe Swift interfaces with comprehensive error handling
  • Automatic CSRF token handling for secure operations

⚡ Powerful Features

  • Bulk operations for node management
  • Start, stop, suspend, and restart VMs
  • Container lifecycle management
  • Cluster monitoring and statistics

📖 Developer Experience

  • Comprehensive DocC documentation
  • Swift Package Manager integration

📦 Quick Installation

Add ProxmoxKit to your Swift package dependencies:

.package(url: "https://github.com/hevey/ProxmoxKit.git", branch: "main")

💡 Quick Start Example

import ProxmoxKit // Configure your Proxmox connection

let config = ProxmoxConfig(
host: "https://proxmox.example.com:8006",
username: "root@pam",
password: "password"
)

let client = ProxmoxClient(config: config)
try await client.authenticate()
// List all nodes
let nodes = try await client.nodes.list()
// Bulk operations
try await client.nodes.stopAll("pve", timeout: 300)
